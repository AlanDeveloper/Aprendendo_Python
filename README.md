# Aprendendo_Python

Usados: 

- Python
- Psycopg2
- Ambiente Virtual
- pgAdmin III
- PostgresSQL


-- Criação do Ambiente Virtual:
  
virtualenv --python=/usr/bin/python3  'NomeDeUmaPastaSemAspas'

Ativar o ambiente: source 'NomeDeUmaPastaSemAspas'/bin/activate

Desativar o ambiente: deactivate

pip install flask
import flask
